\par\noindent The exam will be 6 questions, 2 hours.\n

\par\noindent The questions will not be more difficult than the homework problems. They will likely be easier.\n

\par\noindent Possible topics for Q1-Q3:
\begin{itemize}
	\item Posets, lattices
	\item Linear extensions
	\item M\"obius functions
	\item Dilworth/Mirsky theorem
	\item Projective planes (\textit{but no finite fields})
\end{itemize}\n

\par\noindent Possible topics for Q4-Q6:
\begin{itemize}
	\item Basic concepts:
	\begin{itemize}
		\item Directed/undirected graphs
		\item Trees
		\item Paths
		\item Bipartite graphs
		\item Cycles
	\end{itemize}
	\item Graph colorings
	\item Chromatic number, chromatic polynomial, acyclic orientations
	\item Matchings (Hall's Marriage Theorem)
	\item Extremal graphs, Turan's Theorem, $K_{2,2}$-avoiding, Cauchy-Schwarz inequality
\end{itemize}\n

\newpage
\par\noindent Topics we will not cover:
\begin{itemize}
	\item Matroids
	\item Geometric lattices
	\item Geometry
	\item Ramsey theorem
	\item Probability and Probabilistic Proofs
	\item Planar graphs
\end{itemize}\n

\par\noindent Important proof techniques:
\begin{itemize}
	\item Induction
	\item Contradiction
\end{itemize}
